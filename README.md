# User Guide

<br>

## Requirements

-   git
-   Node.js
-   npm
-   MySQL

<br>

## Set up

### 1. Clone/Download the Repository

```
git clone
cd TokenPrice-scheduler
```

### 2. Install Dependencies:

```
$ npm install
```

### 3. Start MySQL Server

### 4. Create a Database named "test"

### 5. Set up .env

```
DB_HOST=
DB_PORT=
DB_USER=
DB_PWD=
DATABASE_NAME=
```
